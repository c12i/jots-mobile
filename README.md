# jots-mobile

React Native mobile client for jots

Related repositories:

-   [Web](https://github.com/collinsmuriuki/jots-web)
-   [Desktop](https://github.com/collinsmuriuki/jots-desktop)
-   [API](https://github.com/collinsmuriuki/jots-api)

## Usage

Install

```
npm install
```

Start app locally

```
npm start
```
